# 📱 PhoneShop – Responsive Phone Store UI

A modern and responsive phone shopping website built using **HTML, CSS, and JavaScript**.

This project allows users to:

* Browse phones dynamically
* Search phones by brand/model
* Filter by categories and brands
* Add phones to wishlist ❤️
* Add phones to cart 🛒
* View detailed phone specifications
* Experience smooth responsive UI on mobile, tablet, and desktop

---

## 🚀 Features

### 🔍 Search Functionality

Users can search phones by:

* Brand name
* Phone model
